## 4 月趋势观察

1. **Claude Code 生态全面铺开**：从 4/1 首次上榜到 4/8 出现 Karpathy Skills 配套，再到 4/11 / 4/16 的 claude-mem 持久记忆、open-agents 部署、GenericAgent 自演化框架 —— 一个月内 Claude Code 从"终端工具"演化成完整生态。
2. **端侧 / 本地 AI 崛起**：Google AI Edge Gallery、LiteRT-LM、MLX-VLM、Goose、Qwen Code、DeepSeek-TUI 等密集出现，端侧 LLM 部署从概念走向产品化。
3. **AI Agent 平台化**：Hermes Agent、Archon、Rowboat、Multica、GenericAgent、open-agents 等多个 Agent 平台项目接连上榜，Agent 正从"工具"向"团队 / 平台"演进。
4. **自托管需求依然旺盛**：4/14 整期都是开源替代品（Coolify、AppFlowy、NocoDB、Supabase），Self-hosted 仍是 2026 年主旋律之一。
5. **中文 AI 项目破圈**：Qwen Code、Hello-Agents、Andrej Karpathy Skills 中文翻译版等中文社区主导的项目开始频繁上榜，开源生态的多极化趋势明显。
